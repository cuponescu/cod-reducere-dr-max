# Cod Reducere Dr Max

O colecție de coduri de reducere Dr Max. Le folosim pentru testarea cuvintelor cheie [cod reducere Dr Max](https://cuponescu.ro/magazin/dr-max), [voucher Dr Max](https://cuponescu.ro/magazin/dr-max), [cupon Dr Max](https://cuponescu.ro/magazin/dr-max), și [cod promotional Dr Max](https://cuponescu.ro/magazin/dr-max) de pe Cuponescu.ro.

## Instalare

Instalează `cod-reducere-dr-max` prin NPM:

```bash
npm install cod-reducere-dr-max
```

## Utilizare

Pachetul conține un array de obiecte reprezentând coduri de reducere.

În Node:

```javascript
var codes = require('cod-reducere-dr-max')

console.log(codes)

// [
//   {
//     site: 'https://www.drmax.ro',
//     cod_reducere: 'LIFTACTIV',
//     reducere: '25% reducere',
//     descriere: 'Reducere de 25% la produsele Vichy Liftactiv'
//   },
//   ...
// ]
```

## Despre

Cod-reducere-dr-max a fost creat de echipa de la [Cuponescu](https://cuponescu.ro/) pentru a ajuta cu testarea.
